# BookXchange Project
> *Buy, exchange, and donate books with ease!*
---
## What is BookXchange?

It is a web-based classified listing platform that connects book-lovers to streamline the exchange of books. <br>
It is being developed by student enthusiasts under the leadership of *Rui Pinheiro* and *Prof. Goncalo Rijo*
